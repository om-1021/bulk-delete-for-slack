# Privacy Policy — Free Bulk Delete for Slack

_Last updated: 2026-06-09_

This extension does not collect, store, transmit, or sell any personal data.

- **Everything runs locally in your browser.** The extension reads your existing Slack
  session token from the Slack page solely to call Slack's own API to list and delete
  **your own** messages, at your explicit request.
- **No data is ever sent to the developer or any third party.** There are no analytics,
  no tracking, no remote servers, and no remote code.
- The **only** network requests the extension makes are to Slack's own API on
  `*.slack.com`.
- Local browser storage is used only to remember your last-used filter settings on your
  own device.

Because the extension only ever deletes the messages you yourself posted, and deletion is
permanent, please use it deliberately.
